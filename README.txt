# وب‌چت PWA (لینک ثابت)
1) فایل `config.example.js` را به `config.js` تغییرنام بده و مقادیر Firebase پروژه‌ات را داخلش بگذار.
2) محتوا را در یک ریپوی GitHub قرار بده و GitHub Pages را فعال کن.
3) لینک پابلیش‌شده را به دوستان بده؛ همه وارد همان اتاق ثابت خواهند شد.
نکته: برای اتصال مطمئن روی برخی شبکه‌ها، بعداً می‌توانی سرور TURN اضافه کنی.
